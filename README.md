# 🍕 Food Delivery Web Application

A full stack food delivery website built with React JS, 
Node.js, Express, MongoDB & Stripe Payment Gateway.

## 🛠️ Tech Stack
### Frontend
- React JS
- React Router DOM
- Context API
- Axios
- CSS3

### Backend
- Node.js
- Express JS
- MongoDB & Mongoose
- JWT Authentication
- BCrypt
- Multer (Image Upload)
- Stripe Payment

### Tools Used
- Postman (API Testing)
- MongoDB Atlas (Cloud Database)
- Unsplash (Food Images)
- Git & GitHub

## ✨ Features

### User Features
- ✅ User Registration & Login
- ✅ JWT Authentication
- ✅ Browse Food by Category
- ✅ Add to Cart & Update Quantity
- ✅ Secure Stripe Payment
- ✅ Order Verification System
- ✅ My Orders & Order Tracking
- ✅ Fully Responsive Design

### Admin Features
- ✅ Add New Food Items
- ✅ Upload Food Images
- ✅ View All Food Items
- ✅ Delete Food Items
- ✅ Manage Orders
- ✅ Update Order Status
